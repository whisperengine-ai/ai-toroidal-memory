🎯 RECOMMENDATIONS FOR USE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Use GPU:
  ✅ Grid size ≥ 512×512
  ✅ Need real-time performance (< 10ms)
  ✅ Running multiple diffusion steps
  ✅ Throughput > 100M cells/sec required

Use CPU:
  ✅ Grid size < 100×100
  ✅ Simplicity preferred over speed
  ✅ GPU unavailable or development phase

Hybrid:
  ✅ Adaptive selection based on workload
  ✅ Feature flag: #[cfg(feature = "gpu")]
  ✅ Runtime selection possible
